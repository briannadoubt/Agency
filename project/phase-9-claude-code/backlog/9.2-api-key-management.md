# 9.2 API Key Management

Summary:
Implement secure storage and retrieval of Anthropic API key using macOS Keychain.

Acceptance Criteria:
- [ ] Implement `ClaudeKeyManager` using Security framework for Keychain access
- [ ] Add Settings UI for entering, viewing (masked), and updating API key
- [ ] Validate key format (starts with `sk-ant-`)
- [ ] Securely pass key to subprocess via environment variable
- [ ] Handle missing/invalid key errors with user-friendly messages
- [ ] Add "Test Connection" button to verify key validity

Notes:
- Never store API key in UserDefaults or plaintext files
- Use `kSecClassGenericPassword` for Keychain storage
- Service name: `com.briannadoubt.Agency.anthropic-api-key`
- Environment variable: `ANTHROPIC_API_KEY`
