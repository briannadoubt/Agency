# 10.3 Card Pipeline Selection

Summary:
Allow users to select and view flow pipelines for individual cards, with context menu integration and progress visualization.

Acceptance Criteria:
- [x] Pipeline picker in card detail view
- [x] Context menu "Run with Pipeline..." option on card list
- [x] Show current pipeline stage for in-flight cards (e.g., "2/4: implement")
- [x] Pipeline badge/indicator on card in board view
- [x] Ability to cancel running pipeline from card

Notes:
FlowPipeline enum defines: implementOnly, implementThenReview, researchThenImplement, fullPipeline. FlowPipelineOrchestrator tracks active executions.
