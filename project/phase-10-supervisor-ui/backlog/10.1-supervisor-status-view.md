# 10.1 Supervisor Status View

Summary:
Add a status indicator to the main UI showing supervisor state, active runs, and quick controls for start/stop/pause.

Acceptance Criteria:
- [ ] SupervisorStatusView component showing current state (stopped/running/paused)
- [ ] Display active run count and queue depth
- [ ] Start/Stop/Pause buttons that call AgentSupervisor methods
- [ ] Status updates in real-time via @Observable
- [ ] Integration into main toolbar or sidebar

Notes:
The SupervisorCoordinator exposes `status`, `getStatusSnapshot()`, and control methods (`startCoordinator`, `stopCoordinator`, `pauseCoordinator`, `resumeCoordinator`) through AgentSupervisor.
